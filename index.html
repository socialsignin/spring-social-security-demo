spring-social-security-demo by socialsignin

Spring Social Security Demo

Showcase of the spring-social-security module

Dependencies

Spring 3.1.0.RELEASE Spring Security 3.1.0.RELEASE Spring Social 1.0.1.RELEASE Spring Social Security 1.0.1-SNAPSHOT

Install

1. Clone the spring-social-security-demo project locally (eg. git clone git@github.com:socialsignin/spring-social-security-demo.git ) 2. Create API client accounts for Twitter and Facebook, and note the client id and secret for each account. 3. Edit the src/main/resources/environment.properties file, adding the client id and secrets for Facebook and Twitter 4. Run mvn jetty:run from the command line from the base directory of your cloned project.
